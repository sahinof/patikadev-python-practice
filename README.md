## Veri Yapilari ve Algoritmalar - Proje 1

array: [22,27,16,2,18,6]

### 1: Insertion sort asamalari
- [22,27,16,2,18,6] -> 0
- [22,27,16,2,18,6] -> 1
- [16,22,27,2,18,6] -> 2
- [2,16,22,27,18,6] -> 3
- [2,16,18,22,27,6] -> 4
- [2,6,16,18,22,27] -> 5

### 2: Big O gösterimi
O($$n^2$$)

### 3: Time complexity
- Average case: aranan sayi ortada -> O($$n^2$$)
- Worst case: aranan sayi sonda -> O($$n^2$$)
- Best case: aranan sayi basta -> O(n)
 
### 4: Siralandiktan sonra 18'in eklenmesi durumu
